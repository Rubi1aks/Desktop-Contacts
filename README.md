# Desktop Contacts - телефонная книга на Qt6/C++

[![C++](https://img.shields.io/badge/C++-17-blue.svg)](https://isocpp.org/)
[![Qt](https://img.shields.io/badge/Qt-6%2F5-green.svg)](https://qt.io/)

## 📱 О проекте

Десктопное приложение для управления контактами (телефонная книга) с графическим интерфейсом на Qt Widgets.

### Возможности
- ✅ **CRUD**: добавление, просмотр, редактирование, удаление контактов
- ✅ **Таблица** с сортировкой по любому столбцу (клик по заголовку)
- ✅ **Поиск** контактов по любому полю (отдельное диалоговое окно)
- ✅ **Валидация**:
  - Имя/Фамилия/Отчество (только буквы, дефис, пробел)
  - Телефон (форматы: `+7(XXX)XXX-XX-XX` или `8XXXXXXXXXX`)
  - Email (стандартный формат)
  - Дата рождения (YYYY-MM-DD, не будущая)
- ✅ **Хранение** в текстовом файле `contacts.txt` (UTF-8)
- ✅ **Адаптивный интерфейс**: растягивание столбцов, скруглённые кнопки

## 🛠 Технологии
- **C++17**
- **Qt6** (или Qt5 с минимальными правками)
- **CMake** (сборка)
- **QStandardItemModel** (модель таблицы)
- **QSortFilterProxyModel** (фильтрация)

## 📦 Установка и запуск

- CMake ≥ 3.5
